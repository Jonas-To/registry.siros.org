What is this Registry?

This registry hosts credential metadata that describes how to display and validate verifiable credentials across multiple formats:

  • SD-JWT VC - IETF SD-JWT Verifiable Credentials (VCTM)
  • mso_mdoc - ISO 18013-5 mobile documents (mDL, etc.)
  • W3C VCDM 2.0 - W3C Verifiable Credentials Data Model

Credential metadata includes display properties (names, colors, logos), claim definitions, rendering templates, and validation schemas.
