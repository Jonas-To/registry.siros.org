What is VCTM?

Verifiable Credential Type Metadata (VCTM) is a JSON document that describes how to display and validate SD-JWT Verifiable Credentials. It includes:

  • Display properties - Name, description, and visual styling
  • Claims metadata - Human-readable names for credential claims
  • Rendering hints - SVG templates and styling instructions
  • Schema information - JSON Schema for validation
