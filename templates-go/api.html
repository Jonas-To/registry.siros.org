API Reference

Base URL

{{.BaseURL}}/api/v1

GET /schemas

Returns all registered attestation schemas as unsigned JSON (or JWS compact serialization when signing is enabled).

curl {{.BaseURL}}/api/v1/schemas.json

The response contains a data array of SchemaMeta objects, plus total, limit, and offset fields.

GET /schemas/{schemaId}

Returns a single SchemaMeta object by UUID.

curl {{.BaseURL}}/api/v1/schemas/{uuid}.json

Signature Verification

When signing is enabled, responses are also available as .jwt files (JWS compact serialization). Public keys are published as a JWKS at {{.BaseURL}}/api/v1/.well-known/jwks.json.
