What is this Registry?

This registry implements a read-only subset of the ETSI TS11 Catalogue of Attestations specification. It hosts credential metadata describing how to display and validate verifiable credentials across multiple formats:

  • SD-JWT VC — IETF SD-JWT Verifiable Credentials (VCTM)
  • mso_mdoc — ISO 18013-5 mobile documents
  • W3C VCDM 2.0 — W3C Verifiable Credentials Data Model

All API responses are JWS-signed (RFC 7515). Verify signatures using the published JWKS.
